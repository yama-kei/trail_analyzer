Analyze it!

Submit log data in GPX format and Trail Route Analytics let you analyze your climbing/trail running/speed hiking activity.

Tool only accepts GPX formatted file at a momemt. You can export GPX file out from your GPS device/application(e.g. 'Export' menu of MyTracks), or download from various websites where route data is available.

Please note that tool does not support GPX file containing multiple days/set of activities, so you should split your activities accordingly.
